# fofascan
新一代fofa爬虫工具
### 使用限制条件：
需要拥有fofa会员账号
### 使用说明：
1.需先自行登录fofa账号，把获取到的cookie值填入脚本中：_fofapro_ars_session  
2.需填写查询关键词，脚本里有注释  
3.需填写结果存放地址，脚本里有注释
